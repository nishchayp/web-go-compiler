Go Compiler

A partial compiler(lexical analyzer) with error reporting and error recovery for Go(Golang) developed using C, Flex & Bison.

  • The compiler is based on a slightly minified version of the grammar from the official Go documentation.
  • Actual Go compiler, lexer and fmt tool work together to automatically inserts/removes semicolons, line breaks and other such symbols to regularize the code. Please note this compiler(for now) does not account for these optimizations and follows the defined grammar strictly.
